## Windows系統下的配置方式

1. 先去官網安裝 node.js

2. 打開cmd 輸入指令

	npx create-next-app@latest sqv-staking-nft

	npm install ethers @walletconnect/web3-provider web3modal date-fns

	cd sqv-staking-nft
	
	cd pages

	npm install react react-dom ethers date-fns web3modal @walletconnect/web3-provider

	npm run dev
